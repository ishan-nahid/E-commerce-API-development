# E-Commerce Backend System

## Features

- JWT-based User Authentication
- Product & Category Management
- Shopping Cart System
- Order Management

## Tech Stack

- **Language**: Python
- **Web Framework**: FastAPI
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy
- **Authentication**: JWT
- **Password Hashing**: Passlib
